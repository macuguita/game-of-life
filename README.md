# Conway's Game of Life

## About

A implementation of Conway's game of life in C using raylib, this is mainly a test for raylib and using shaders.

## Quick start

### Macos
```console
$ ./build_macos.sh
$ ./build/game_of_life
```

### Linux
```console
$ ./build_linux.sh
$ ./build/game_of_life
```

### Windows
```console
$ .\build_msvc.bat
$ .\build\game_of_life.exe
```
